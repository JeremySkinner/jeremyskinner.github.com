FluentValidation 2.0 beta 1 is now available for download. This release contains the following changes:

  • Integration with ASP.NET MVC 3 Beta
  • Windows Phone 7 compatibility
  • Performance improvements to AttributedValidatorFactory
  • Bug fixes

I’ve also put together NuPack packages for FluentValidation 1.3. From within the Package Manager console you can use the command Add-Package FluentValidation to install the core FluentValidation package for either .NET or Silverlight 4, while typing Add-Package FluentValidation.MVC2 will install FluentValidation’s ASP.NET MVC 2 integration as well as the core package.

A FluentValidation 2.0 package will also be available via NuPack once it’s closer to release.
