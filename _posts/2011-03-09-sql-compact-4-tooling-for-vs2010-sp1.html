--- layout: post title: "SQL Compact 4 tooling for VS2010 SP1" postId: 636 ---

If you’ve installed VS2010 SP1 and you can’t see any tooling for working SQL Server Compact 4.0, make sure that the “Microsoft Visual Studio 2010 SP1 Tools for SQL Server Compact 4.0” package is installed via the Web Platform Installer.

This can be found under the “Products” section then under “Database”.
